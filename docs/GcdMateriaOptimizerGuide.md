# GCD 魔晶石自动镶嵌使用指南

本文介绍如何在网页应用内使用“GCD 自动镶嵌”功能，根据目标 GCD 秒数自动生成最高期望伤害的镶嵌方案。

## 前置条件
1. **选择职业与等级**：在页面顶部选择当前的职业和等级，确保与游戏内一致。
2. **装备与食物**：在装备列表中为每个部位选择装备，并在食物面板中选择将要食用的食物。未完整选择装备或食物会导致无法正确计算。
3. **确定目标 GCD**：根据想要的技能速度或咏唱速度阈值，确定目标 GCD 秒数（如 2.46、2.40 等）。

## 操作步骤
1. 打开右侧的“魔晶石”面板，切换到 **“GCD 自动镶嵌”** 标签页。
2. 在输入框中填写目标 GCD（单位：秒）。
3. 点击 **“一键生成”** 按钮。
4. 观察提示：
   - 如果显示“已套用方案：GCD X.XX，期望伤害 YYY”，表示已经根据目标 GCD 自动完成镶嵌。
   - 如果提示无法满足目标 GCD，请确认全部装备和食物已选择，并适当放宽目标值。

## 计算逻辑概览
- 系统会先重置当前装备的魔晶石等级，按槽位的可用等级从高到低尝试。
- 优先使用速度类魔晶石（战技/咏唱）降低 GCD，直到达到或略高于目标阈值。
- 在保持 GCD 不超出目标的前提下，用剩余槽位依次尝试暴击、直击、信念/技速等候选属性，以期望伤害提升为准择优选择。
- 计算过程中会忽略任何无效快照（如伤害或 GCD 计算为 NaN），确保结果可靠。

## 常见问题
- **总是提示无法满足目标 GCD**：请检查是否遗漏了某个装备部位或食物，或目标 GCD 过低导致在当前装备等级下无法达到。
- **结果 GCD 比目标更快**：当速度槽位足够强时，可能轻微低于目标值，但算法会优先保持不低于目标；如需更宽松范围，可提高目标值后重试。
- **想恢复手动镶嵌**：重新在魔晶石面板中手动调整各槽位即可覆盖自动方案。
